Local Business Sales Project¶

Project Overview¶

This project analyzes sales data from a local business to uncover insights on revenue trends, location performance, and customer behavior. The dataset was transformed from wide to long format and cleaned to ensure consistency.

Main Objectives:

  • Calculate KPIs such as total revenue and average order value (AOV).
  • Identify top-performing locations and months.
  • Visualize trends for better decision-making.

Step 3: Clean and Merge Datasets¶

To prepare our data for analysis, I perform the following preprocessing steps:

  • Standardize column names: Strip whitespace, convert to lowercase, and replace spaces with underscores
  • Convert date columns: Ensure sale_date is in datetime format for later time-based analysis
  • Merge datasets: Join sales_df and price_df on the shared item column
  • Create a new metric: Calculate total_revenue as quantity × unit_price
  • Preview results: Confirm merged and cleaned dataset using .head()

# Convert sale_date column to datetime (if exists)
if 'sale_date' in sales_df.columns:

# Merge the two datasets if there's a shared column (like 'item')

    merged_df = sales_df.merge(price_df, on='item', how='left')

    merged_df['total_revenue'] = merged_df['quantity'] * merged_df['unit_price']

Step 5: Reshape and Enrich Data¶

To transform our cleaned data into a format suitable for analysis, I performed the following:

  • Reshape wide to long: Used pd.melt() to convert product columns into rows with item and quantity fields
  • Drop missing values: Removed rows where quantity was NaN (no purchase occurred)
  • Convert data types: Converted quantity and price columns to numeric types using pd.to_numeric()
  • Standardize text: Cleaned the item column in both DataFrames to match (stripping and lowercasing)
  • Merge price info: Merged price_df into reshaped data using the item column as key
  • Debug merge: Verified shape and missing values after merge to troubleshoot mismatches

I encountered some NaN prices during the merge due to mismatched item names, which will be addressed in the next step.

# --- Step 1: Reshape the wide format into long format ---

    var_name='item',            # new column for item name
    value_name='quantity'       # new column for quantity sold
)
